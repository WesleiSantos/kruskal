# Ambiente Python para Projeto de Grafos e Geolocalização

Este projeto utiliza Python para trabalhar com grafos e geolocalização. As principais bibliotecas usadas são `networkx`, `matplotlib` e `geopy`. Siga as instruções abaixo para configurar o ambiente e executar o projeto.

---

## **Requisitos**
- Python 3.6 ou superior
- `pip` (gerenciador de pacotes do Python)

---

## **Configuração do Ambiente**
Execulte os seguintes comandos no terminal para instalar as dependencias:
```bash
pip install networkx
pip install matplotlib
pip install geopy
```

## Execução
Para execultar o código basta digitar o comando no prompt:
```bash 
python3 graph.py
```
